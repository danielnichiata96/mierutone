# Claude Configuration for PitchLab JP

Este diretório contém configurações e habilidades do Claude para trabalhar com o projeto PitchLab JP.

## Estrutura

- `config.json`: Configuração principal do projeto
- `skills/frontend-skills.md`: Habilidades e padrões de frontend

## Como Usar

Ao trabalhar no projeto, o Claude deve:
1. Consultar `config.json` para entender a stack
2. Seguir os padrões definidos em `skills/frontend-skills.md`
3. Manter consistência com componentes existentes

## Atualizações

Quando novos padrões ou componentes são adicionados, atualize os arquivos de skills correspondentes.
